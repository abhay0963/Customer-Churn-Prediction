# Models Folder

This folder will contain the following files after running `src/train_model.py`:

- churn_model.pkl
- scaler.pkl
- feature_names.json
- model_results.csv
- confusion_matrix.png
- feature_importance.png

These files are not committed to version control (see .gitignore) since they
are generated artifacts, not source code. Run the training script once to
generate them locally.
